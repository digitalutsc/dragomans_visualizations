Json Visualization examples

Specification details:
  • w (width): Width of graph (gray box)
  • h (height): Height of graph (gray box)
  • maxLabelWidth: Maximum distance horizonatally between nodes (as well as the limits of the graph)
  • minRadius: Minumum radius of nodes
  • transitionDuration: Time for animation to take, in milliseconds
  • scalingFactor: Change in size for each node layer (I think)

Another specification for each of these graphs is overflow-x, which is currently set to scroll. This means when the graph is wider than the graph limits (the gray box), you will be able to scroll to see the rest of the graph.

This can also be set to auto, which simply expands the graph's limit. (It can look a bit wierd if it extends way off to the right hand side of the page) To change this, go into styles.css and change the overflow-x property in .graph.
